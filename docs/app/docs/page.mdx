# 简介

**Nextra** 是构建在 Next.js 之上的框架，可帮助您构建以内容为核心的网站。它继承了 Next.js 的所有优秀特性，并额外提供了轻松创建基于 Markdown 内容的强大能力。

## 常见问题

Nextra 常见问题解答收集了关于该项目的有用问答。如果您的问题在此未得到解答，请[发起讨论](https://github.com/shuding/nextra/discussions)。

<details>
  {/* prettier-ignore */}
  <summary>我能在 Next.js `pages` 路由中使用 Nextra 吗？</summary>
Nextra 4 仅支持 `app` 路由。只有 Nextra 1/2/3 版本支持 `pages` 路由。

</details>

<details>
  <summary>Nextra 能否与 X 一起使用？</summary>
大多数情况下答案是"可以"。由于 Nextra 只是 Next.js 的一个插件，几乎所有能在 React 中实现的功能都可以在 Nextra 中使用。以下是一些示例和指南：

- [使用 Tailwind CSS](/docs/guide/tailwind-css)
- [使用自定义 CSS 和 Sass](/docs/guide/custom-css)
- [使用自定义字体](https://nextjs.org/docs/basic-features/font-optimization)

</details>

<details>
  {/* prettier-ignore */}
