# 内置组件

<Callout type="info">
  本 API 参考文档由兜底路由文件 `/docs/built-ins/[name]/page.tsx` 自动生成，使用了全新的 [Nextra `<TSDoc />` 组件](/docs/built-ins/tsdoc)。
</Callout>

Nextra 包含多个内置组件，可用于优化内容样式：

{/* `pageMap` prop will be removed once Nextra will add dynamic routes in pageMap */}

<OverviewPage
  filePath={metadata.filePath}
  pageMap={
    (await getEnhancedPageMap())
      .find(o => o.name === 'docs')
      .children.find(o => o.name === 'built-ins').children
  }
  icons={{
    WarningIcon: GitHubWarningIcon,
    IdCardIcon,
    FolderTreeIcon,
    OneIcon,
    TableIcon,
    CardsIcon
  }}
/>
