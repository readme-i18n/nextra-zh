# Mermaid

Nextra 支持 [mermaid](https://mermaid.js.org) 图表功能。与 GitHub 类似，您可以在 Markdown 文件中通过使用 `mermaid` 代码块语言来创建图表。Nextra 默认采用 [`@theguild/remark-mermaid`](https://npmjs.com/package/@theguild/remark-mermaid) 包，该包会将代码块替换为 `<Mermaid>` 组件。

## 示例

<Demo />

export async function Demo() {
  const mermaidCodeblock = `\`\`\`mermaid
graph TD;
subgraph AA [Consumers]
A[Mobile app];
B[Web app];
C[Node.js client];
end
subgraph BB [Services]
E[REST API];
F[GraphQL API];
G[SOAP API];
end
Z[GraphQL API];
A --> Z;
B --> Z;
C --> Z;
Z --> E;
Z --> F;
Z --> G;
\`\`\``
  const rawJs = await compileMdx(`${mermaidCodeblock}
## Usage
~~~md filename="Markdown"
${mermaidCodeblock}
~~~
`)
  return <MDXRemote compiledSource={rawJs} components={{ Mermaid }} />
}
