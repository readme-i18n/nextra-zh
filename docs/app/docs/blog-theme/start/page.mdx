## 作为新项目启动

<Steps>
### 安装

要手动创建 Nextra 博客站点，您需要安装 **Next.js**、**React**、**Nextra** 以及 **Nextra Blog Theme**。在项目目录中运行以下命令安装依赖项：

```sh npm2yarn
npm i next react react-dom nextra nextra-theme-blog
```

> [!NOTE]
>
> 如果项目中已安装 Next.js，只需额外安装 `nextra` 和 `nextra-theme-blog` 作为附加组件即可。
