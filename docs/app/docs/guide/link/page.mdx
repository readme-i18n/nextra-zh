# Next.js 链接

所有相对路径的 Markdown 链接都会自动转换为
[Next.js 链接](https://nextjs.org/docs/app/building-your-application/routing/linking-and-navigating)。
这意味着目标页面会被预加载。当您点击链接时，
页面会像[单页应用(SPA)](https://en.wikipedia.org/wiki/Single-page_application)一样
在客户端加载，无需完整页面刷新。例如：

```md filename="Markdown"
Click [here](/about) to read more.
```

等价于：

```mdx filename="MDX"
import Link from 'next/link'

Click <Link href="/about">here</Link> to read more.
```

此特性使得 Nextra 页面间的导航快速且无缝。
