# TypeScript

Nextra 基于 TypeScript 构建，并提供了开箱即用的优秀 TypeScript 支持。本指南将帮助您在 Nextra 项目中充分发挥 TypeScript 的优势。

## 快速开始

要在 Nextra 项目中使用 TypeScript，您需要：

<Steps>
### 将 TypeScript 和类型包安装为 `devDependencies`

```sh npm2yarn
npm i -D typescript @types/react @types/node
```

### `tsconfig.json`

您可以在项目根目录手动创建 `tsconfig.json` 文件，或者将现有文件的扩展名改为 `.ts` 或 `.tsx`，Next.js 会自动检测项目中的 TypeScript 并为您生成 `tsconfig.json` 文件。

</Steps>

## 类型定义

Nextra 为其组件和配置提供了分发代码的类型定义。您可以通过将主题配置文件重命名为 `.ts` 或 `.tsx` 扩展名并导入主题配置类型来利用这些类型，例如对于 `nextra-theme-docs`：

```tsx filename="theme.config.tsx"
import type { DocsThemeConfig } from 'nextra-theme-docs'

const config: DocsThemeConfig = {
  // Your theme configuration
}
export default config
```

通过在 Nextra 项目中使用 TypeScript，您可以及早发现错误、提高代码质量，并通过更好的自动补全和类型推断来提升开发体验。
