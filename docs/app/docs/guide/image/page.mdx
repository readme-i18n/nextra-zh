# Next.js 图片组件

在 MDX 中使用 [Next.js Image](https://nextjs.org/docs/app/getting-started/images) 的标准方式是直接导入该组件：

```mdx filename="MDX"
import Image from 'next/image'

<Image src="/demo.png" alt="Hello" width={500} height={500} />
```

## 静态图片

> [!NOTE]
>
> 该功能默认通过 Nextra 配置中的 `staticImage: true` 启用。

Nextra 支持通过 Markdown 语法自动优化静态图片导入。您无需指定图片的宽高，只需使用 `![]()` Markdown 语法：

```md filename="Markdown"
![Hello](/demo.png)
```

这会加载 `public` 文件夹内的 `demo.png` 文件，并自动用 Next.js `<Image>` 组件进行包裹。

> [!TIP]
>
> 如果您不想通过 `public` 托管图片，也可以使用 `![](../public/demo.png)` 从相对路径加载图片。

使用 Next.js Image 时，不会出现布局偏移，且在图片加载时会默认显示精美的模糊占位图：

![Nextra](/opengraph-image.jpeg)

## 图片缩放

> [!NOTE]
>
> 图片缩放功能默认全局启用。

在默认配置下，如需使用此功能，只需通过 `![]()` Markdown 语法插入图片。

### 禁用图片缩放

对于 `nextra-docs-theme` 和 `nextra-blog-theme`，您可以通过替换 MDX 中使用的 `img` 组件来禁用图片缩放功能。

```jsx filename="theme.config.jsx"
import { Image } from 'nextra/components'

export default {
  // ... your other configurations
  components: {
    img: props => <Image {...props} />
  }
}
```

### 为特定图片启用/禁用缩放功能

当缩放功能**全局禁用**时，若需对特定图片启用缩放，可使用 `<ImageZoom>` 组件：

```mdx
import { ImageZoom } from 'nextra/components'

<ImageZoom src="/demo.png" />
```

当缩放功能**全局启用**时，若需对特定图片禁用缩放，直接使用 `<Image>` 组件即可：

```mdx
import { Image } from 'nextra/components'

<Image src="/demo.png" />
```
