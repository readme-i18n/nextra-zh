在您的 `package.json` 中添加以下脚本：

```json filename="package.json"
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start"
},
```

> [!TIP]
>
> 您可以通过在 `dev` 命令后添加 `--turbopack` 标志来启用开发环境下的 Turbopack：
>
> ```diff filename="package.json"
> - "dev": "next",
> + "dev": "next --turbopack",
> ```

根据您使用的包管理器，可以通过以下命令启动开发服务器：

```sh npm2yarn
npm run dev
```

或使用生产模式：

```sh npm2yarn
npm run build
npm run start
```

> [!NOTE]
>
> 如果您不熟悉 Next.js，请注意开发模式会显著较慢，因为 Next.js 会编译您访问的每个页面。

### 添加 Next.js 配置

在项目根目录下创建 `next.config.mjs` 文件，内容如下：

```js filename="next.config.mjs"
import nextra from 'nextra'

// Set up Nextra with its configuration
const withNextra = nextra({
  // ... Add Nextra-specific options here
})

// Export the final Next.js config with Nextra included
export default withNextra({
  // ... Add regular Next.js options here
})
```

通过此配置，Nextra 将处理 Next.js 项目中的 Markdown 文件。更多 Nextra 配置选项，请查阅[指南](/docs/guide)。
