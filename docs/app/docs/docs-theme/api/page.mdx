# API

## `useThemeConfig` 钩子

`useThemeConfig` 钩子返回你的
[主题配置](/docs/docs-theme/theme-configuration) 值，用于
动态配置项目。

```js
import { useThemeConfig } from 'nextra-theme-docs'
```

## `useConfig` 钩子

```js
import { useConfig } from 'nextra-theme-docs'
```

`useConfig` 钩子返回当前页面上下文中的数据。
