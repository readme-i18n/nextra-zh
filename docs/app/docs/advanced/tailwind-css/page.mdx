# Tailwind CSS

Tailwind CSS 是一个 CSS 框架，提供一组预定义的 CSS 类来快速为元素添加样式。

<Steps>

## 遵循官方指南

按照官方提供的 [Next.js 版 Tailwind CSS 指南](https://tailwindcss.com/docs/guides/nextjs) 为项目配置 Tailwind CSS。

## 创建 `globals.css` 文件

在 `globals.css` 中导入 Tailwind CSS：

```css filename="globals.css"
@import 'tailwindcss';

/* 可选：导入 Nextra 主题样式 */
@import 'nextra-theme-docs/style.css'; /* 或 nextra-theme-blog/style.css */

@variant dark (&:where(.dark *));
```

## 在根布局中导入样式

如需全局应用样式，请在根布局文件中导入 `globals.css`：

```jsx filename="app/layout.jsx"
import '../path/to/your/globals.css'

export default async function RootLayout({ children }) {
  // ... 您的布局逻辑
}
```

</Steps>
